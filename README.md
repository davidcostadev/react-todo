# React TODO

Esse é um projeto com foco em apredenzado do ReactJs, 

### Instalação
``` bash
$ git clone https://github.com/davidcostadev/react-todo.git
$ cd react-todo 
```

Você poder usar ```npm``` ou ```yarn```

``` bash
$ npm install
```

ou 

``` bash
$ yarn
```

## Comandos

**npm**

``` bash
$ npm run dev
$ npm run build
$ npm run start
```

**yarn**

``` bash
$ yarn dev
$ yarn build
$ yarn start
```
